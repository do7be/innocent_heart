●はじめに
・この度は当ソフトウェアをダウンロードしていただき
  まことにありがとうございます。

●注意
本ソフトウェアは「ゆとり世代」には推奨されていません。
「ゆとり世代」の方は難易度にご注意下さい。


●Innocent Heartの概要
・本ソフトウェアは横スクロールアクションゲームです。
・剣や魔法を駆使してダンジョンの奥深くへと進み、魔王を倒してください。


●ストーリー　※重要なので必ずお読みください
昔々、あるところにまー助という男がいました。
彼は剣と魔法に長けていましたが、姿がとても醜かったのです。
ある日、まー助は「ジェイヨン」という国に立ち寄りました。
なにやら慌しかったので、人々に聞いてみると
「姫が魔王にさらわれた。すでに何人か向かったが、皆死んでしまった」
とのことでした。
話を聞いたまー助はすぐさま魔王のいるというダンジョンへと向かったのでした。


●動作環境
・Windows XP,Vista,7 にて動作を確認しております。
・また、Cygwin,glpngのインストールをしてある環境が必須です。


●コンパイル
・Makefileを作成してありますのでCygwinのコマンド上で
　「　　make　　」(鉤括弧なし)
　と入力していただければコンパイルが完了し、j06441.exeという実行ファイルが作成されます。


●実行方法
・コンパイル後、Cygwinのコマンド上で
　「　　./j06441　　」(鉤括弧なし)
　と入力していただければプログラムが実行されます。


●操作方法
カーソルキー：左右移動
Zキー：決定、剣攻撃
Xキー：ジャンプ
Aキー：ファイア
Sキー：アイス
Dキー：サンダー
Spaceキー：ポーズ


●ルール
ダンジョンにはモンスターが徘徊しています。
モンスターにぶつかるか攻撃を受けると１発で死んでしまいます。
コンティニューは何度でも可能です。
ファイア、アイス、サンダーは使用回数の制限があります。
残り使用回数は左上に表示されます。
敵を倒すと魔法の使用回数を回復することができます。


●敵情報
・スライム
　動きは遅く、つきあたりまで止まらない。

・モヒカンリザード
　動きが速く、つきあたりまで止まらない。

・レッドデーモン
　空を飛んでいて、まー助に向かって衝撃魔法を撃ってきます。
　
・ウィザードマン
　動きは遅く、風魔法を撃ってきます。

・魔王
　以下の魔法を唱えてきます。
　-ブルーファイア
　-ダークサンダー
　-ウォーターバルーン
　-メテオ
　ボスのライフは20ですが、剣とサンダーは2ダメージ、その他は1ダメージを与えられます。
